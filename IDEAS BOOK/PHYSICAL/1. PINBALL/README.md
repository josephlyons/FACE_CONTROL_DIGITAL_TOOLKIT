Parts
-

- Lasercut Parts
- MiniSero 180 Servo Motors
- Arduino Uno Rev 3
- M4 30mm Bolts & Nuts

Assembly
-
1. Cut parts A-D.
2. Insert bolts as Figure 1 shows.
3. Mount servos to back.
4. Superglue flippers (D) to servo arms.
5. Wire servos to arduino.
6. Install & run softwares via Github.
